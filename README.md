# CityGuideApp
Android app for discovering tourist attractions, restaurants, shopping places and travel safety information.

# City Guide
An Android application built using Kotlin and Jetpack Compose.

## Features

- Search cities
- Tourist attractions
- Famous restaurants
- Local shopping places
- Scam awareness
- Travel tips

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Room Database
- Firebase

## Status

Project planning phase.
